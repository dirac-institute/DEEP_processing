# Processing

## Biases
Make survey biases:
```
$ sbatch processing/stampede/bias.sh
```

Missing bias:
```
$ bash processing/stampede/missing_bias.sh
```

## Flats

Make survey flats:
```
$ sbatch processing/stampede/flat.sh
```

Missing flat:
```
$ bash processing/stampede/missing_flat.sh
```

## DRP

```
$ sbatch processing/stampede/drp.sh
```
